Olá, esse é o meu portfólio! Aqui você vai encontrar um pouco dos meus trabalhos com dados, relacionados principalmente com as áreas de análise e engenharia de dados.

Projetos Recentes

Entendendo o churn em clientes de serviços bancários

(Python / Scikit-learn / XGBoost) O projeto tem como base o dataset “Predicting Churn for Bank Customers” - disponibilizado pelo Kaggle - e realiza uma extensa análise exploratória de dados para entender o que leva os clientes a cancelarem os serviços prestados por um banco e aplicação de modelos de Machine Learning para “prever” potenciais casos de churn. No que você deve ficar de olho Análise exploratória de dados gerando vários insights sobre os clientes do banco e identificando um perfil dentre aqueles que deram churn; Aplicação e tuning automatizado dos 3 modelos de ML escolhidos: Regressão Logística, Random Forest e Extreme Gradient Boost Classifier; Interpretação das métricas de avaliação do desempenho dos modelos de ML; Entrega: novo dataset com probabilidade de churn de cada cliente do dataset de teste.

Minha primeira API

(Python / Flask / SQL / Heroku) Nesse projeto foi construída uma API que retorna dados salariais dos profissionais da área de economia (minha área de formação) no Brasil em 2019. A API foi construída baseada em Flask, um framework web muito utilizado para construir aplicações web de forma rápida e utilizando Python. Os dados foram coletados de forma automática através um script shell diretamente de uma fonte na internet; passou-se então para a limpeza dos dados através principalmente da biblioteca Pandas; os dados tratados foram armazenados em um banco de dados relacional (SQLite); a API foi construída em Flask com os endpoints, o método aceito (“GET” apenas) e as consultas em SQL que retornarão os dados nos endpoints; por fim, foi realizado o deploy da API no Heroku.
